# Maghreb AI Trip Planner

## Overview

Maghreb AI Trip Planner is a web application designed to help users plan their trips to Morocco using AI-powered itinerary generation. The application leverages modern web technologies such as React and Vite, providing a seamless user experience with features like background slideshows, testimonials, and service offerings.

## Features

- **AI-Powered Trip Planning**: Generate personalized travel itineraries based on user preferences.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **User Authentication**: Secure sign-up and login functionality.
- **Dynamic Backgrounds**: Engaging background slideshows to enhance user experience.
- **Testimonials**: User feedback showcased to build trust and credibility.
- **Service Offerings**: Detailed descriptions of services provided by the application.

## Technologies Used

- **Frontend**: 
  - React
  - Vite
  - Tailwind CSS
  - Framer Motion
  - Radix UI
  - Axios for API calls

- **Backend**: 
  - Node.js
  - Express
  - MongoDB (with Mongoose)
  - Firebase for authentication
  - Google Generative AI for trip planning

## Installation

To get started with the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/maghrebaitripplanner.git
   cd maghrebaitripplanner
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   - Create a `.env` file in the root directory and add your Firebase and Google API keys.

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## Usage

- **Sign Up / Login**: Users can create an account or log in to access personalized features.
- **Create a Trip**: Users can input their travel preferences, including origin, destination, dates, and budget.
- **View Generated Trip**: After submitting their preferences, users can view a detailed itinerary generated by the AI.
